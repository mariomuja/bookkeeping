# Profile Images

## Adding Your Photo

To add your profile photo to the login page:

1. **Add your photo** to this directory with the name: `mario-muja.jpg`
2. **Recommended specifications**:
   - Format: JPG or PNG
   - Size: 100x100 pixels (or larger, will be automatically resized)
   - File size: Under 500KB for optimal loading

If no photo is provided, a placeholder with your initials "MM" will be displayed.
