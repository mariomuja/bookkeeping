# Screenshots

This folder contains screenshots of the main features of International Bookkeeping.

## How to Create Screenshots

1. Start the application with `.\start-app.ps1`
2. Open http://localhost:4200 in your browser
3. Login with demo/demo123
4. Navigate to each section and take screenshots:

### Required Screenshots:

1. **dashboard.png** - Dashboard page with financial metrics
   - Navigate to: http://localhost:4200/dashboard
   - Shows: Total Assets, Liabilities, Equity, Net Income, Activity Summary

2. **framework-import.png** - Chart of Accounts with Framework Import Modal
   - Navigate to: http://localhost:4200/accounts
   - Click "Import Framework" button
   - Shows: Modal with SKR03, SKR04, IKR, GAAP, IFRS, PCG, PGC frameworks

3. **chart-of-accounts.png** - Chart of Accounts Table
   - Navigate to: http://localhost:4200/accounts
   - Shows: List of all accounts with numbers, names, types, categories

4. **journal-entries.png** - Journal Entries List
   - Navigate to: http://localhost:4200/journal-entries
   - Shows: Transaction list with debit/credit accounts, amounts, status

5. **reports.png** - Reports Page with DATEV Export
   - Navigate to: http://localhost:4200/reports
   - Shows: Trial Balance, Balance Sheet, P&L cards, DATEV Export section

6. **documentation-modal.png** - Documentation System
   - Click the "?" icon in the header
   - Shows: Documentation index categorized by topic

## Tips for Good Screenshots

- Use a resolution of at least 1920x1080
- Ensure the browser window is maximized
- Wait for all content to load before taking the screenshot
- Use browser developer tools (F12) to take high-quality screenshots
- Or use tools like Snipping Tool (Windows), Screenshot (Mac), or browser extensions
